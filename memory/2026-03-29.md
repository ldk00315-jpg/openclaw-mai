## OpenClaw移行手順 (Zorin OS 18 → Linux Mint)
とんすけ先輩がZorin OS 18からLinux Mintへの移行を考えている際の、OpenClawエージェントデータを保持する手順：
1. バックアップ: `cp -r ~/.openclaw ~/backup-openclaw`
2. 必要に応じてsystemdサービスもバックアップ: `cp /etc/systemd/system/openclaw*.service ~/backup-services`
3. Linux Mintをクリーンインストール（重要データは別途バックアップ）
4. Node.js v22以上をインストール
5. OpenClawを再インストール: `curl -fsSL https://openclaw.ai/install.sh | bash`
6. 初期設定: `openclaw onboard --install-daemon`
7. データ復元:
   - `cp -r ~/backup-openclaw ~/.openclaw`
   - `sudo chown -R $USER:$USER ~/.openclaw`
   - systemdサービスファイルを復元して再有効化
8. 動作確認: `openclaw` コマンドで起動確認
